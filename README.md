# ecommerce-return-rate-ml-dashboard
Business Intelligence and Machine Learning Dashboard for E-commerce Return Rate Analysis using Power BI

1) Project Overview
This project analyzes e-commerce business performance and integrates a Machine Learning model to predict product return risk.
The dashboard provides business KPIs, revenue trends, return rate analysis, and ML-based risk insights using Power BI.

2) Objectives
Analyze revenue and order performance
Measure return rate trends
Identify high-risk orders using ML prediction
Perform category and state-level analysis
Integrate business intelligence with predictive analytics

3) Tools Used
Python (for ML model)
Pandas & Scikit-learn
Power BI
GitHub

4) Key Insights
Total Revenue: 13M+
Return Rate: ~16%
~15% orders predicted as high-risk
Revenue shows declining trend over time
Certain states contribute significantly higher revenue
